# Rahul Naskar — Portfolio

A dark, minimal portfolio built with **Vite + React + TypeScript**.

## 🚀 Local Development

```bash
npm install
npm run dev
```

## 🏗️ Build

```bash
npm run build
```

## ▲ Deploy to Vercel

### Option 1 — Vercel CLI
```bash
npm i -g vercel
vercel
```

### Option 2 — GitHub Integration
1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com) → **New Project**
3. Import your repo
4. Vercel auto-detects Vite — click **Deploy**

That's it. No extra config needed (`vercel.json` handles SPA routing).

## ✏️ Customise

| File | What to edit |
|------|-------------|
| `src/components/Hero.tsx` | Name, bio, links |
| `src/components/Skills.tsx` | Skill groups & tags |
| `src/components/Contact.tsx` | Email, GitHub, LinkedIn |
| `src/index.css` | Color variables (`--accent`, `--bg`, etc.) |
